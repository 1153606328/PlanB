使用pythond的一些记录
### 1、基本
#### 列表 (数组)
- append 末尾添加
- pop 末尾删除
- remove 删除指定值
- del 删除指定元素
- insert 指定位置添加\
    排序:
    - sort 顺序
    - sorted 临时顺序
    - reverse 翻转  

#### 字符串
- title( )  首字母大写
- upper( )  全大写
- lower( )  全小写  
  删除空白
    - rstrip( ) 结尾
    - lstrip( ) 开头
    - strip( )  两端
